<COMM.Dll 사용법>
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. CMD Folder
  - DBContext.cs
    ※ 절대 수정 금지.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2. Interfaces Folder
  - IDataBase.cs
    -> DB관련 명령어.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
3. Models Folder
  - DefaultContext.cs
    ※ 절대 수정 금지.
  - etc.cs
    -> 연결된 DB에 대한 Model 클래스.
       ※ DB 파일에 대한 갱신이 필요할 경우 "Scaffold-DbContext "Data Source=D:\DB\Default.db" Microsoft.EntityFrameworkCore.Sqlite -OutputDir "sqlite/Models/" -StartupProject Comm -Force"
         해당 명령어를 복사하여, 패키지 관리자 콘솔에 기본프로젝트 "Comm" 선택 후, 명령어를 붙여 넣으면 자동으로 Model 클래스들이 갱신된다.
         필요없는 Model 클래스는 수동으로 삭제해줘야된다.
